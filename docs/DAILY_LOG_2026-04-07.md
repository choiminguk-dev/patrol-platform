## 시간순 작업 흐름

### 2) 현황탭 달력 팝업
- 날짜 클릭 → 월 단위 달력 → 일 선택 → 즉시 이동
- 일/토 색상 구분, 미래 비활성화, 오늘 링 표시

### 3) 보고서 4종 UI 개선 (모바일)
- 동 현장 점검 일지: 테이블 → 【순찰 사항】【주민 소통】【기타 특이사항】 섹션
- 일일 순찰일지: 순번|일시|내용 테이블 → 섹션 기반
- 주간 순찰보고: 다음 주 계획 삭제 + 섹션 통일
- 실적 종합보고: 개선사항 삭제 + 섹션 통일
- 동장 이름 삭제 / 순찰자 란 삭제
- AI 오타 후처리: 정소→청소

### 4) 참여인원 풀 추론 전체 카테고리 매핑

- 이후 프롬프트 강화: 주소판 절대 plates 제외

### 7) GPS 역지오코딩 (반복 개선) ★
| 단계 | 방식 | 결과 |
|------|------|------|
| 1차 | Nominatim (OSM) | "후암동"만 |
| 2차 | Kakao Map (coord2address) | road_address null 이슈 |
| 3차 | 지번→도로명 재검색 | "후암동 244-1" 폴백 |
| 4차 | 9좌표 그리드 병렬 | 도로명 적중 (32-7) |
| 5차 | 5m+10m 2단계 + EXIF GPS 우선 | 정확도 향상 (32-6) |
| 6차 | 카메라 방향(heading) 5m 전진 | **카메라가 향하는 건물 매핑** |

- `src/lib/geocode.ts` (Kakao + Nominatim 폴백)
- `src/app/api/geocode/route.ts` (클라이언트 호출용)
- TrackA: GPS 감지 즉시 자동 주소 입력 + UI 표시
