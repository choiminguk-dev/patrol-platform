## 04-09 후반 작업

### 1) TrackB 병합 모드 AI 병합 추가 (3af9a29)
- 기존: 병합 모드 = 체크박스 수동 선택만
- 추가: "🤖 AI 병합" 버튼 → 같은 zoneId 그룹 자동 묶음
- 후속 개선 (a7b0ca0): zone 없어도 같은 주소(공백 무시 비교) 그룹도 머지
  - `cat:{category}|addr:{normalized}` 키로 그룹화
  - 다른 카테고리는 절대 머지 안 함

### 2) 후보 구역 알림 + 지도 배지 + AI 자동 제안 (afddfca)
- **NEW 배지**: localStorage `lastSeenCandidates` 추적 → 마지막 확인 이후 새 후보 시 빨간 NEW 배지
- **지도 미리보기**: 각 후보 카드에 "🗺 지도" 배지 → 클릭 시 forward geocode → 지도 모달
- **AI 자동 장소특징**: "구역으로 추가" 클릭 시
  - 해당 카테고리×주소의 최근 entry에서 AI reasoning 추출
  - landmark 필드 자동 입력 (사용자는 검토만)
  - `/api/zones/candidates` 응답에 `suggestedLandmark` 포함

### 3) #2 상세 편집 — 인라인 수정 (2cc283b)
- **DetailModal 확장** (`src/app/admin/dashboard/page.tsx`)
  - 메모: 텍스트영역 + onBlur 자동 저장
  - 주소: 텍스트영역 + onBlur 자동 저장
  - 카테고리: select 변경 즉시 저장
  - 사진: 편집 모드 토글 → 각 사진 X 버튼으로 삭제
- **사진 추가**: photoAddRef 파일 input → uploadPhotosChunked → 기존 photoUrls에 append
- **권한**: ADMIN 또는 본인만 수정 가능 (PATCH `/api/entries/[id]`)
- **PATCH 확장**: category, memo, addressText, photoUrls, zoneId 동적 SET 절 생성

### 4) 현황탭 일자별 입력 AI 병합 (befc88e)
- 기존: TrackB(분석 직후)에서만 AI 병합
- 추가: Dashboard 일자별 입력 화면에 "🤖 AI 병합" 버튼
- 신규 API: `POST /api/entries/auto-merge { date }`
  - 같은 날짜의 모든 entry 조회
  - zoneId 우선, 없으면 카테고리+정규화 주소로 그룹화
  - 첫 entry로 사진 합치고 나머지 삭제
  - 응답: `{ mergedGroups, removedEntries }`

### 5) PhotoViewer 좌우 스와이프 (464a511)
- 기존: 더블탭 줌 + 핀치 줌 + 드래그 팬
- 추가: 단일 터치 스와이프 → 사진 이동
- swipeStart ref (x, y, time) → touchend에서 판정
  - 가로 변위 > 50px, 세로 < 60px, 시간 < 600ms

- 줌 상태에서는 스와이프 비활성 (드래그 팬과 충돌 방지)

### 7) 일시 포맷 + _미리보기.html (f503ad6)

**일시 포맷 변경**
- `시간: 22:25:16` → `일시: 4.9. 22:25` (월.일. HH:MM, 초 제거)
- `_info.txt`, `_목록.csv` 헤더 동시 적용
- 폴더명은 그대로 `{seq}_{HHMMSS}_{주소}` (충돌 방지용 초 단위 키 필요)

**_미리보기.html 동봉** ⭐
- ZIP 루트에 단일 HTML 파일 (자체 완결, 외부 의존 없음)
- 더블클릭 → 기본 브라우저로 모든 항목 카드 형태 표시
- 각 카드:
  - 좌측 메타: 연번 배지 + 일시/위치/구역/분류/담당/메모
  - 우측 사진 그리드 (160px+ aspect 1:1)
  - 사진 클릭 → 원본 새 탭 열림
  - "복사" 버튼 → 일시/위치/메모 클립보드 복사 (hwpx 셀 paste용)
- 한글 폴더/파일 `encodeURIComponent` (file:// 호환)
- 모바일 반응형 (720px 이하 세로 정렬)
- clipboard API 폴백: `document.execCommand('copy')`

**해결한 문제**
- Windows Explorer 폴더별 보기 모드 따로 저장 → 사진 폴더라도 `_info.txt` 때문에 "일반 항목" 템플릿으로 잡혀 미리보기 안 켜짐
- HTML 미리보기로 Explorer 의존성 자체 제거

| **AI 병합 (Dashboard)** | 일자별 입력 | 날짜 전체 | 같은 zone/주소 자동 합침 | DB에 들어간 후 사후 정리 |
