## 자동화 로드맵 — "업로드 → 끝" 달성 전략

### 현재 병목
```
업로드 30장 (~15초) → 블러 확인 (~30초) → AI 분석 (~20초) → 수동 정정 (~10분) → 제출
                                                              ↑ 여기가 병목
```

### Phase 1: 주소판 힌트 (구현 → 롤백, face-api hang으로 보류)
- 클라이언트에서 파란색 주소판 자동 감지 → AI에 "사진 N번이 주소판" 힌트
- 10장 안에 주소판 2~3개 → AI가 정확히 분할
- **예상 효과**: 수동 정정 10분 → 2~3분

### Phase 2: 자동 제출 모드 (미구현)
- zone 매칭된 그룹 (주소 일치 + 참조사진 매칭) → **자동 제출**
- 미매칭 그룹만 사용자 리뷰
- **예상 효과**: 30장 → 6개 그룹 중 4개 자동 제출, 2개만 확인

### Phase 3: 사진 순서 기반 그룹 보정 (미구현)
- 촬영 패턴: 작업사진 → 주소판 → 작업사진 → 주소판...
- 주소판 감지 결과(Phase 1) + 사진 순서 → 그룹 경계 자동 보정
- 서버 후처리에서 "주소판 뒤의 사진은 다음 그룹" 규칙 적용
- **예상 효과**: AI 오그룹핑 자동 수정 → 수동 분할 거의 불필요

### 최종 목표
```
업로드 30장 → 블러 확인 → AI 분석 → 자동 제출 (확인만)
~15초          ~30초        ~20초       ~30초
총 약 1분 30초 (현재 약 12분)
```
